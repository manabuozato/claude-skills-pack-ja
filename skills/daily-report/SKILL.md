# 日報自動化スキル / Daily Report Generator

## Description

今日の活動を自動的にまとめ、日報フォーマットで出力するスキルです。
カレンダーやミーティング記録と連携して、効率的に日報を作成します。

**トリガー**: 「日報」「daily report」「日報更新して」「今日の日報」

---

## 処理手順

### 1. 今日の日付を確認

- 本日の日付を取得
- 曜日を含めて記載

### 2. カレンダーの確認（MCP連携時）

Google Calendar MCPが有効な場合:
- 今日のイベント一覧を取得
- 各ミーティングの概要を取得

MCP未連携の場合:
- ユーザーに今日の主な予定・活動を入力してもらう

### 3. ミーティング記録の統合

今日のチャット内でミーティングメモや議事録を作成していた場合:
- それらの内容を日報に統合
- 決定事項とアクションアイテムを反映

### 4. フォーマット整形

以下の形式で出力します。

```
━━━━━━━━━━━━━━━━━━━━━━━━
日報 / Daily Report
━━━━━━━━━━━━━━━━━━━━━━━━

■ 日付: YYYY年M月D日（曜日）
■ 氏名: [ユーザー名]

━━━━━━━━━━━━━━━━━━━━━━━━

【本日の実績】

▼ ミーティング
- [HH:MM] [会議名] — [概要・結果]

▼ タスク
- [タスク名] — [進捗・成果]

▼ その他
- [その他の活動]

━━━━━━━━━━━━━━━━━━━━━━━━

【明日の予定】
- [予定1]
- [予定2]

【所感・気づき】
[今日の振り返り・学びなど]

【懸案事項】
- [懸念があれば記載]

━━━━━━━━━━━━━━━━━━━━━━━━
```

## カスタマイズポイント

- `「英語で」` → 英語フォーマットで出力
- `「簡潔に」` → 箇条書きのみ（所感省略）
- `「Slackに貼れる形で」` → Slack用フォーマット
- `「Notionに」` → Notion DB更新（MCP連携時）

## MCP連携パターン

### Google Calendar + 日報

```
「今日のカレンダーを確認して日報を作って」
→ カレンダーから自動でミーティング一覧を取得し、日報に反映
```

### Notion + 日報

```
「日報をNotionに保存して」
→ Notion の日報データベースに新しいエントリを作成
```
