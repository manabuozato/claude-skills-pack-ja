# 議事録生成スキル / Meeting Minutes Generator

## Description

会議の文字起こしテキストから、構造化された日本語議事録を自動生成するスキルです。

**トリガー**: 「議事録」「minutes」「打ち合わせの内容整理して」「ミーティングメモ」

---

## 処理手順

### 1. 入力の受け取り

ユーザーから文字起こしテキストが提供されたら、以下の形式を自動判定します:
- Tactiq形式（`[HH:MM:SS] 話者名: 発言`）
- otter.ai形式（`Speaker N タイムスタンプ`）
- Google Meet形式（`話者名 HH:MM`）
- Zoom形式（VTT風タイムスタンプ）
- その他（自由形式のメモ）

### 2. 話者の名寄せ

表記揺れを統一します。
- 姓のみ / フルネーム / ニックネーム → 統一表記
- 同姓の場合はフルネームで区別

### 3. 敬語の正規化

デフォルトは「だ・である調」。
- 「おっしゃっていました」→「〜と述べた」
- 「させていただきます」→「〜する」
- 「〜でございます」→「〜である」
- 「〜かと思います」→「〜の見解を示した」

オプション: 「ですます調で」と指定があれば丁寧語で統一。

### 4. 構造化と抽出

- 時系列をトピック別に再構成
- 決定事項を抽出（「〜に決まりました」「〜で行きましょう」）
- アクションアイテムを抽出（「〜さん、お願いします」「〜までに」）
- 持ち帰り検討 → アクションアイテムとして記録

### 5. 出力

```
■ 会議名: [会議名]
■ 日時: [YYYY年M月D日（曜日）HH:MM-HH:MM]
■ 場所: [場所]
■ 参加者: [参加者リスト]

【要約】
[3行以内]

【議題1: [議題名]】
- [発言者]: [要点]

【決定事項】
1. [決定事項]

【アクションアイテム】
| 担当 | 内容 | 期限 |
|------|------|------|

【次回予定】
```

## カスタマイズポイント

- `「ですます調で」` → 丁寧語で出力
- `「簡潔に」` → 要約重視
- `「和暦で」` → 令和表記
- `「バイリンガルで」` → 日英併記
- `「Notionに貼れる形で」` → Markdown形式
