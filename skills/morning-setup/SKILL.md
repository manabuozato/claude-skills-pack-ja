# 朝のセットアップスキル / Morning Setup Routine

## Description

毎朝の業務開始時に、今日の予定・メール・タスクを一括確認し、1日のフォーカスを設定するスキルです。

**トリガー**: 「朝」「セットアップして」「今日のtodo」「おはよう」「モーニングルーティン」

---

## 処理フロー

### ステップ1: 日付の更新

- 本日の日付・曜日を表示
- 今週の残り日数を表示
- 月末までの残り日数（締切意識の喚起）

### ステップ2: カレンダー確認

Google Calendar MCPが有効な場合:
- 今日の予定一覧を取得
- 各予定の時刻・場所・参加者を表示
- 準備が必要な予定にフラグ

MCP未連携の場合:
- 「今日の予定を教えてください」と聞く

### ステップ3: メール確認

Gmail MCPが有効な場合:
- 未読メールの件数と要約
- 緊急性の高いメールをハイライト
- 返信が必要なメールのリスト

MCP未連携の場合:
- 「気になるメールがあれば貼り付けてください」と案内

### ステップ4: タスク整理

Notion MCPが有効な場合:
- TODOデータベースから今日のタスクを取得
- 優先度順にソート

MCP未連携の場合:
- 「今日やることを教えてください」と聞く

共通処理:
- タスクをアイゼンハワーマトリックスで分類
  - A: 緊急×重要 → 最優先
  - B: 重要×非緊急 → 時間を確保
  - C: 緊急×非重要 → 委任検討
  - D: 非緊急×非重要 → 削除検討

### ステップ5: 今日のフォーカス設定

```
━━━━━━━━━━━━━━━━━━━━━━━━
おはようございます！
YYYY年M月D日（曜日）のセットアップ
━━━━━━━━━━━━━━━━━━━━━━━━

📅 今日の予定
- [HH:MM] [予定名] @ [場所]
- [HH:MM] [予定名] @ [場所]
  ⚠️ 準備: [準備内容]
- [HH:MM] [予定名]

📧 メール（未読: X件）
- 🔴 [緊急] [送信者]: [件名]
- 🟡 [要返信] [送信者]: [件名]
- ⚪ [参考] [送信者]: [件名]

✅ 今日のタスク（優先度順）
- [A] [タスク名] — [期限]
- [A] [タスク名] — [期限]
- [B] [タスク名]
- [C] [タスク名]

🎯 今日のフォーカス
「[今日最も重要な1つのこと]」

💡 ひとこと
[簡単な応援メッセージや、今日の予定に関するアドバイス]

━━━━━━━━━━━━━━━━━━━━━━━━
```

## カスタマイズポイント

- `「英語で」` → 英語で出力
- `「簡潔に」` → 予定とタスクのみ
- `「フォーカスだけ」` → タスク優先度分析のみ
- `「明日の準備も」` → 明日の予定のプレビューを追加
