# 朝のセットアップスキルの使用例

## 入力

ユーザーが「おはよう、今日のセットアップして」と指示。

（Google Calendar MCP と Gmail MCP が有効な場合の例）

## 出力

```
━━━━━━━━━━━━━━━━━━━━━━━━
おはようございます！
2026年2月27日（木）のセットアップ
━━━━━━━━━━━━━━━━━━━━━━━━

📅 今日の予定（4件）
- [10:00-11:00] プロジェクトX定例 @ 会議室A
  ⚠️ 準備: 先週のアクションアイテムの進捗確認
- [12:00-13:00] ランチMTG 佐藤さん @ カフェテリア
- [14:00-15:00] 提案書レビュー @ Zoom
- [16:00-16:30] 週次1on1 上司 @ オンライン

📧 メール（未読: 8件）
- 🔴 [クライアントA] 納品スケジュールの確認依頼
- 🟡 [佐藤] 明日の会議資料の共有
- 🟡 [人事部] 勤怠確認のお願い
- ⚪ その他5件（ニュースレター、通知等）

✅ 今日のタスク（優先度順）
- [A] 提案書の最終版完成 — 今日中
- [A] クライアントA への返信 — 午前中
- [B] 来週の会議アジェンダ作成
- [B] バグチケット #234 の確認
- [C] 経費精算の提出

🎯 今日のフォーカス
「提案書を仕上げてクライアントに送る」

💡 ひとこと
午前中にクライアントへの返信と提案書作業に集中し、
午後のレビューに備えましょう。ランチMTGでリフレッシュも忘れずに。

━━━━━━━━━━━━━━━━━━━━━━━━
```
