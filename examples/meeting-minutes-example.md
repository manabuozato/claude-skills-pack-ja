# 議事録スキルの使用例

## 入力（文字起こし抜粋）

```
[00:00:10] 田中: それでは定例を始めます。本日はウェブサイトリニューアルの進捗確認です。
[00:00:20] 佐藤: よろしくお願いします。デザインの件ですが、トップページのワイヤーフレームが完成しました。
[00:00:35] 田中: ありがとうございます。レビューはいつ頃できそうですか。
[00:00:42] 佐藤: 明日中にレビュー依頼を出します。鈴木さんにも確認いただきたいです。
[00:00:55] 鈴木: 了解しました。金曜日までにフィードバックを返します。
[00:01:05] 田中: では、トップページのワイヤーフレームレビューは金曜日までということで。
```

## 出力（生成された議事録）

```
■ 会議名: ウェブサイトリニューアル 定例会議
■ 日時: 2026年2月27日（木）
■ 参加者: 田中、佐藤、鈴木

【要約】
ウェブサイトリニューアルの進捗確認を実施した。
トップページのワイヤーフレームが完成し、レビュースケジュールを確認した。

【議題1: デザイン進捗】
- 佐藤: トップページのワイヤーフレームが完成した。
- 佐藤: 明日中にレビュー依頼を出す予定。
- 鈴木: 金曜日までにフィードバックを返す。

【決定事項】
1. トップページのワイヤーフレームレビューを金曜日までに完了する

【アクションアイテム】
| 担当 | 内容 | 期限 |
|------|------|------|
| 佐藤 | レビュー依頼の送付 | 2/28（金） |
| 鈴木 | ワイヤーフレームのフィードバック | 2/28（金） |
```

## ポイント

- 敬語（「いただきたいです」「了解しました」）が「だ・である調」に変換されている
- 散漫な会話が「議題→決定→アクション」に構造化されている
- 期限が明確に抽出されている
